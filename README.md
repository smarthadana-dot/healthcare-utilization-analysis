# Healthcare Utilization & Revenue Trend Analysis

This project evaluates healthcare service utilization patterns across inpatient, outpatient, and emergency department settings. The objective was to analyze patient volume trends and revenue distribution using structured SQL queries and statistical analysis in R.

## Tools Used
- SQL for data aggregation
- R for statistical analysis and visualization
- Excel for data validation
- GitHub for version control

## Project Focus
- Aggregation of patient counts and revenue by service category
- Quarterly trend analysis
- Revenue per patient evaluation
- Visual communication of healthcare operational metrics

This repository demonstrates structured data handling, reproducible analysis workflow, and health-focused data visualization.
